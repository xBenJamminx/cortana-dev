# SEO Audit Checklist

Complete criteria for analyzing content SEO performance.

## On-Page SEO Factors

### Title Tag (Weight: High)
- [ ] Contains primary keyword (preferably near beginning)
- [ ] 50-60 characters (avoid truncation in SERPs)
- [ ] Compelling/clickable (not just keyword-stuffed)
- [ ] Unique across site

### Meta Description (Weight: Medium)
- [ ] Contains primary keyword naturally
- [ ] 150-160 characters
- [ ] Includes call-to-action or value proposition
- [ ] Accurately summarizes content

### URL Structure (Weight: Medium)
- [ ] Contains primary keyword
- [ ] Short and readable (3-5 words ideal)
- [ ] Uses hyphens, not underscores
- [ ] No unnecessary parameters or numbers

### Header Structure (Weight: High)
- [ ] Single H1 containing primary keyword
- [ ] H2s break content into logical sections
- [ ] H3s used for subsections (not skipping levels)
- [ ] Headers form scannable outline
- [ ] Secondary keywords in H2s where natural

### Keyword Optimization (Weight: High)
| Location | Requirement |
|----------|-------------|
| First 100 words | Primary keyword appears |
| Body content | 1-2% density (natural usage) |
| Image alt text | Keyword in at least one image |
| Headers | Primary in H1, secondary in H2s |

### Content Quality (Weight: Critical)
- [ ] Matches search intent completely
- [ ] More comprehensive than competing pages
- [ ] Original insights/data/examples
- [ ] No thin sections (every H2 has substance)
- [ ] Updated/accurate information
- [ ] Proper grammar and spelling

### Readability (Weight: Medium)
- [ ] Flesch-Kincaid grade level appropriate for audience
- [ ] Short paragraphs (2-4 sentences)
- [ ] Varied sentence length
- [ ] Active voice dominant
- [ ] Scannable (headers, bullets, bold)

### Internal Linking (Weight: Medium)
- [ ] Links to relevant internal pages (3-5 minimum for long content)
- [ ] Descriptive anchor text (not "click here")
- [ ] Links placed contextually
- [ ] No orphan pages created

### External Linking (Weight: Low-Medium)
- [ ] Links to authoritative sources for claims
- [ ] Opens in new tab (user preference)
- [ ] No broken links
- [ ] Balanced (not excessive)

### Media Optimization (Weight: Medium)
- [ ] Images compressed (<100KB when possible)
- [ ] Descriptive file names (not IMG_001.jpg)
- [ ] Alt text on all images
- [ ] Video transcripts provided if applicable

## E-E-A-T Assessment

### Experience
- [ ] First-person insights present
- [ ] Real-world examples included
- [ ] Practical application shown
- [ ] Author has used/tested what they write about

### Expertise
- [ ] Technical accuracy verified
- [ ] Appropriate depth for topic
- [ ] Industry terminology used correctly
- [ ] Complex concepts explained clearly

### Authoritativeness
- [ ] Author byline with credentials
- [ ] Citations to authoritative sources
- [ ] Original research/data if possible
- [ ] Expert quotes or references

### Trustworthiness
- [ ] Balanced perspective (addresses counterarguments)
- [ ] Transparent about limitations
- [ ] Contact information accessible
- [ ] No misleading claims

## Search Intent Checklist

### Informational Intent
- [ ] Comprehensive coverage of topic
- [ ] Clear explanations
- [ ] Visual aids (diagrams, examples)
- [ ] FAQ section if appropriate
- [ ] Related topics linked

### Transactional Intent
- [ ] Clear product/service information
- [ ] Pricing visible (if applicable)
- [ ] Trust signals (reviews, guarantees)
- [ ] Strong, clear CTA
- [ ] Easy conversion path

### Commercial Intent
- [ ] Comparison information
- [ ] Pros/cons clearly stated
- [ ] Multiple options presented
- [ ] Decision-making framework
- [ ] Links to transactional pages

## Scoring Guide

| Score | Description |
|-------|-------------|
| 9-10 | Exceptional. Best-in-class, likely to rank. |
| 7-8 | Strong. Minor optimizations needed. |
| 5-6 | Average. Significant opportunities missed. |
| 3-4 | Weak. Major issues affecting rankability. |
| 1-2 | Poor. Needs fundamental restructuring. |

Calculate overall score as weighted average:
- Content Quality: 30%
- Keyword Optimization: 25%
- Structure (headers, title, meta): 20%
- E-E-A-T: 15%
- Technical (links, media): 10%
