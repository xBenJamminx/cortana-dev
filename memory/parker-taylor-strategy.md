# Parker & Taylor — Cash Engine
**Updated: Feb 16, 2026**

## Purpose
Generate near-term income with predictable conversion. Fastest path to revenue. Reduces stress. Buys runway.

## Financial Context
- Income: ~$11.5k/mo (existing clients)
- Expenses: ~$15k/mo
- Monthly burn: ~$3.5k
- Savings in crypto (volatile, down)
- Particle job uncertain
- Dependents: 3-year-old + infant

## What Parker & Taylor Sells

### Offer 1: 30-Day System Build
- Fixed scope, defined workflow
- 30-day delivery, acceptance criteria
- **$6k–$8k target**
- Examples: Deal flow ingestion, KPI dashboard automation, client onboarding automation, CRM restructuring, reporting automation

### Offer 2: Automation Partnership
- Ongoing structured engagement
- **$4k–$6k/month**
- Multiple workflow implementation
- Reliability hardening + iterative improvement

## Ideal Buyer Profile
Defined by traits, NOT company size:
- High manual ops load
- High-value operator time
- Resistance to hiring full-time ops
- Cross-tool chaos
- Reporting burden
- Repetitive data movement

**Segments:** Family offices, independent sponsors, small PE/VC, real estate operators, agencies, founder-led services

## Acquisition Strategy
**No X dependency.** Channels: LinkedIn outbound, email outbound (Apollo enriched), hiring intercept, growth-signal targeting.

## Weekly Outbound Requirements
Per outbound block (min 2 blocks/week):
- 15 new touches
- 10 follow-ups

Follow-up cadence: Day 0 → Day 3 → Day 7 → Day 14 → Park.

## Sales Process
1. 15-minute diagnostic call
2. Identify one bottleneck
3. Recommend either build or partnership
4. Send one-page proposal within 24 hours
5. Start with onboarding checklist

## Onboarding Checklist
- [ ] Access list
- [ ] Current workflow map
- [ ] Success metric defined
- [ ] Approval path confirmed
- [ ] Error escalation path defined
- [ ] Timeline agreed

## Weekly Non-Negotiables
- 2 outbound blocks minimum
- 1 delivery block minimum
- 1 proposal or diagnostic call per week target

## Guardrails
- No scope creep
- 30-day builds tightly defined with acceptance criteria
- Partnerships structured with clear deliverables
- Proposal sent within 24 hours of diagnostic call

## Success Metrics
**Weekly:** New touches, replies, calls booked, proposals sent, deals closed
**Monthly:** Revenue added, revenue retained, hours saved estimate delivered
