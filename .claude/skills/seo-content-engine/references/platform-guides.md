# Platform-Specific Guides

Optimization rules for each major platform.

## LinkedIn

### Algorithm Priorities
- Dwell time (time spent reading)
- Early engagement (first 60-90 minutes critical)
- Comments > reactions > shares
- Native content over external links

### Format Rules
- **Hook:** First 1-2 lines must stop scroll (visible before "see more")
- **Length:** 800-1,200 characters optimal for engagement
- **Line breaks:** Single sentences with white space between
- **No links in post:** Put links in first comment (improves reach)
- **Hashtags:** 3-5 relevant hashtags at end

### High-Performing Content Types
1. Personal stories with business lessons
2. Contrarian takes on industry norms
3. Tactical how-tos with specific steps
4. Career advice and lessons learned
5. Behind-the-scenes of business decisions

### LinkedIn Post Template
```
[Hook - one powerful line]

[Space]

[Context/Story - 2-3 short paragraphs]

[Space]

[Key insight or lesson]

[Space]

[Takeaway or question for engagement]

[Space]

#hashtag1 #hashtag2 #hashtag3
```

### Avoid
- Corporate jargon ("synergy," "leverage," "ecosystem")
- Humble brags disguised as lessons
- Posting links in main post body
- Walls of text without line breaks
- Engagement bait ("Comment YES if you agree!")

## Twitter/X

### Algorithm Priorities
- Replies and quote tweets
- Bookmarks (strong signal)
- Time on tweet/thread
- Profile visits from tweet

### Format Rules
- **Hook:** First 7-10 words determine engagement
- **Single tweets:** 70-100 characters optimal CTR
- **Threads:** 5-15 tweets, each must standalone
- **Media:** Images/video increase engagement 2-3x
- **Timing:** Post when your audience is active

### Thread Structure
```
1/ [Hook - bold claim or curiosity gap] 🧵

2/ [Context - why this matters now]

3-N/ [One idea per tweet, each valuable alone]

Last/ [Summary + CTA (follow, retweet, bookmark)]
```

### High-Performing Content Types
1. Contrarian takes with receipts
2. Breakdowns of successful examples
3. Curated lists/resources
4. Real-time commentary on events
5. Personal stories with universal lessons

### Avoid
- Threads that could be a single tweet
- Every tweet starting with "Thread:"
- Excessive self-promotion
- Dunking without adding value
- Engagement farming ("Retweet if...")

## Reddit

### Algorithm Priorities
- Upvote velocity (early votes count more)
- Comment volume and quality
- Time spent on post
- Upvote/downvote ratio

### Format Rules
- **Title:** Specific, genuine, not clickbait
- **Body:** Substantial value upfront
- **Tone:** Authentic, not salesy
- **Timing:** Know subreddit peak hours

### Content Strategy
1. **Provide value first** — Reddit hates self-promotion
2. **Match subreddit culture** — Lurk before posting
3. **Be specific** — Vague posts get ignored
4. **Engage in comments** — OP participation boosts visibility
5. **Use proper formatting** — Headers, bullets, TL;DR

### Reddit Post Template
```
[Descriptive, specific title - no clickbait]

**TL;DR:** [2-3 sentence summary for skimmers]

[Context/Background]

[Main content with headers for sections]

[What you learned / what you're asking]

---

Happy to answer questions in comments.
```

### Avoid
- Overt self-promotion (use 10:1 rule: 10 value posts per 1 promotional)
- Clickbait titles (Reddit users punish this)
- Ignoring subreddit rules
- Cross-posting spam
- Generic advice without specifics

## Email (Subject Lines & Preview Text)

### Subject Line Rules
- **Length:** 30-50 characters (mobile-friendly)
- **Personalization:** [Name] increases open rates 20%+
- **Urgency:** Time-sensitive language when genuine
- **Curiosity:** Open loops work well
- **Lowercase:** Can feel more personal/casual

### Subject Line Formulas
| Type | Formula | Example |
|------|---------|---------|
| Curiosity | [Question or incomplete thought] | "the email I almost didn't send" |
| Benefit | [Outcome] in [Timeframe] | "double your conversions this week" |
| Personal | [Casual/conversational tone] | "quick question for you" |
| Urgency | [Time element + benefit] | "24 hours left: your strategy session" |
| Social proof | [Number/Authority] + [Outcome] | "how 1,000 founders handle rejection" |

### Preview Text Rules
- **Length:** 40-90 characters (shows after subject)
- **Complement subject:** Don't repeat, extend
- **Action-oriented:** Hint at value inside
- **Avoid:** Unsubscribe links or boilerplate showing

### Preview Text Formula
```
Subject: [Curiosity gap]
Preview: [What they'll get if they open]

Example:
Subject: "the 3am email that changed everything"
Preview: "Plus: the exact script I used to close the deal"
```

### Avoid
- ALL CAPS or excessive punctuation!!!
- Spam trigger words ("free," "act now," "limited time")
- Misleading subject lines
- Generic subjects ("Newsletter #47")
- Subject/preview mismatch

## Cross-Platform Principles

1. **Native first:** Each platform rewards native content over repurposed
2. **Format for scanning:** Assume readers skim first
3. **Hook in first 1-2 lines:** Visible before any click/tap
4. **One big idea:** Don't dilute with multiple topics
5. **Clear CTA:** Tell readers what to do next
6. **Engage with responses:** Algorithms reward conversation
