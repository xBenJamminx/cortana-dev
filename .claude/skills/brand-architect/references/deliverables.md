# Deliverables Reference

## Brand Guidelines Document Structure

When generating the brand guidelines document (Word format), follow this structure:

### Cover Page
- Brand name (large, in brand font if specified)
- "Brand Guidelines" subtitle
- Version/date

### Table of Contents
- Auto-generated or manual listing of sections

### 1. Brand Overview
**1.1 Brand Story**
- Origin and purpose (2-3 paragraphs)
- What problem the brand solves

**1.2 Mission Statement**
- Single statement of purpose

**1.3 Vision Statement**
- Aspirational future state

**1.4 Core Values**
- 3-5 values with brief descriptions (2-3 sentences each)

### 2. Brand Personality
**2.1 Brand Archetype**
- Primary archetype name and description
- Secondary archetype if applicable
- What this means for how the brand behaves

**2.2 Personality Traits**
- 4-6 key personality adjectives
- "We are X, not Y" contrasts for clarity

### 3. Visual Identity
**3.1 Logo** (placeholder section)
- Note: "Logo designs to be inserted"
- Include guidance on clear space, minimum size, placement

**3.2 Color Palette**
- Primary color with hex, RGB, CMYK values
- Secondary colors with values
- Accent colors with values
- Usage guidelines (when to use each)

**3.3 Typography**
- Primary typeface (heading use)
- Secondary typeface (body use)
- Font weights and hierarchy
- Usage examples

**3.4 Photography & Imagery Style**
- Style description
- Do's and don'ts
- Example descriptions or mood keywords

### 4. Voice & Tone
**4.1 Brand Voice**
- Voice characteristics (3-4 attributes)
- Explanation of each attribute

**4.2 Tone Variations**
- Formal contexts (e.g., legal, investor communications)
- Casual contexts (e.g., social media, customer support)
- Marketing contexts

**4.3 Writing Guidelines**
- Words we use / Words we avoid
- Grammar and style preferences
- Example sentences in brand voice

### 5. Messaging Framework
**5.1 Tagline**
- Primary tagline
- Usage guidance

**5.2 Elevator Pitches**
- 15-second version
- 30-second version
- 60-second version

**5.3 Boilerplate Descriptions**
- Short (25 words)
- Medium (50 words)
- Long (100 words)

**5.4 Key Messages**
- 3-5 core messages for different audiences or contexts

### 6. Application Examples (Optional)
- Business card layout concept
- Email signature format
- Social media profile guidance
- Presentation template guidance

## One-Page Brand Summary Structure

A condensed reference for quick alignment. Format as single page with clear visual hierarchy.

### Header
- Brand name (prominent)
- Tagline (below name)

### Left Column (Identity)
**Mission**
- Single statement

**Archetype**
- Primary archetype with 2-3 word description

**Key Differentiators**
- 3 bullet points

### Right Column (Expression)
**Voice Characteristics**
- 3-4 adjectives

**Color Palette**
- Visual swatches with hex codes (primary, secondary, accent)

### Footer
**Elevator Pitch** (15-second version)

## Document Generation Notes

### For Word Documents
- Use consistent heading styles (Heading 1, 2, 3)
- Include page breaks between major sections
- Use tables for color specifications
- Leave placeholder boxes for logo/visual examples
- Include a "Last updated" date field

### Formatting Preferences
- Clean, professional layout
- Generous white space
- Brand colors used as accents in document design
- Page numbers in footer

### File Naming Convention
`[BrandName]_Brand_Guidelines_v[X]_[YYYY-MM-DD].docx`
`[BrandName]_Brand_Summary_[YYYY-MM-DD].docx`

## Alternative Deliverable Formats

### HTML Brand Guide
For web-based, interactive guidelines. Use when:
- Brand will be shared widely via link
- Need for interactive color pickers, copy-to-clipboard
- Frequent updates expected

### Markdown Brand Guide
For developer-friendly, version-controlled documentation. Use when:
- Technical audience
- Integrating with existing documentation systems
- Need for git-based collaboration

### Presentation Deck
For stakeholder presentations or onboarding. Use when:
- Need to present and explain brand
- Training new team members
- Agency handoff
